# YQT BOT

## Features
- User-friendly interaction
- Multiple signal handling 
- Customizable alerts

## Quick Start
1. Clone the repository: `git clone https://github.com/freelancerjob114-oss/YQT-BOT-PRO.git`
2. Navigate to the project directory: `cd YQT-BOT-PRO`
3. Install dependencies: `npm install`
4. Run the bot: `npm start`
